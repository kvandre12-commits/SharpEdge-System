# Section Gap Views

This folder holds phone-facing section-gap view builders and templates.

It may read upstream artifacts such as:
- `ops/section_gap/pilot_config.example.json`
- `ops/section_gap/events/*.json`
- `ops/section_gap/observations/*.json`
- future section-gap dashboard artifacts

It must not redefine section truth outside approved upstream artifacts.
It renders and packages mobile-facing state only.
